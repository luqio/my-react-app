# React-app

#### 介绍
基于 React + TypeScript + Vite 的前端开发模板

#### 技术栈

- **框架**: React + TypeScript + Vite
- **UI 组件**:
  - tc-components - 自定义 UI 组件库
  - arco-design - 企业级 UI 组件
- **工具链**:
  - pnpm - 包管理器
  - ESLint - 代码检查
  - Prettier - 代码格式化


#### 开发教程

1. 下载依赖包：
    ```
    pnpm i 
    ```
2. 启动项目
    ```
    npm run dev
    ```
3. 打包项目
    ```
    npm run build
    ```
